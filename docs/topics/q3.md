# 如何保证平台能保持稳定地存在？
平台能一直活下去是所有运营者最关心的问题，毕竟谁也不想辛辛苦苦搭的平台说没就没。经过我们的调查，在这个问题上，我们需要综合考虑这些因素。

### 能让平台稳定存在的做法
- 技术上多做备份。定期备份数据，多存几个地方，避免服务器炸了所有东西都没了。
- 提前搞定合规问题。该备案的备案，该审核的审核，别因为违规被直接关停。
- 攒够足够的用户。用的人多了，平台就有了存在的价值，不容易随便死掉。
- 把流程写清楚。就算有人走了，后来的人看着文档也能接着干。

### 这样做的麻烦所在
- 费钱又费精力。多备份、搞安全、写文档都需要花时间和钱，学生团队本来就没多少资源。
- 技术门槛有点高。不是所有人都会搞服务器备份、安全防护这些东西。
- 容易变得臃肿。为了稳定加了太多东西，反而让平台越来越难维护。
