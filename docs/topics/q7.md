# 是否需要经费支持？如果有经费支持，要如何保证平台的纯洁性？
没钱寸步难行，但有钱了也可能会变坏。这是很多非盈利平台都会遇到的两难问题。经过我们的调查，在这个问题上，我们需要综合考虑这些因素。

### 有经费支持的优点
能解决基本开销。服务器、域名这些不用自己掏腰包了。
能提升平台体验。可以买更好的服务器，开发更多好用的功能。
能激励团队成员。给大家发点小补贴，干活也更有动力。

### 有经费支持的缺点
容易变味。为了赚钱可能会接乱七八糟的广告，破坏平台原本的氛围。
容易产生矛盾。钱怎么花、分给谁，这些问题很容易引起团队内部吵架。
可能会失去独立性。如果拿了别人的赞助，可能就要听别人的话。

### 保证平台纯洁性的关键做法
钱的进出都公开。每一笔钱从哪来、花到哪去，都清清楚楚写出来给大家看。
只接不影响内容的赞助。绝对不能为了钱让平台变成广告场。
建立多人监督机制。不能一个人说了算，大家一起管钱。
